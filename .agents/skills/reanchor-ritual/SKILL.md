# Reanchor Ritual

Relative path: `.agents/skills/reanchor-ritual/SKILL.md`

Purpose: This ritual wrapper is the normal entrypoint for compressing work into a resumable handoff package in `agent-browser-tdd-relay`. Use it when there will be a pause, a handoff, or growing context cost. It should preserve the trusted planning/history state that future work needs in order to resume cleanly without drift.

## Use when

- there will be a pause or handoff
- prompt context is getting too expensive
- continuity risk is increasing
