# Reanchor Ritual Checklist

Relative path: `automation/checklists/reanchor-ritual-checklist.md`
Project: Agent Bridge

Purpose: This checklist defines the minimum continuity package for pauses, handoffs, or context compression in Agent Bridge. Use it to produce a compact, trusted restart package that includes current state, next action, and the planning and history context needed to resume safely.

- Capture current state in plain language
- Record what is done and what is next
- Record active docs and artifacts to trust
- Record roadmap, backlog, and journal state that matters for restart
- Record drift risks and do-not-forget constraints
- Produce a concise resume package
