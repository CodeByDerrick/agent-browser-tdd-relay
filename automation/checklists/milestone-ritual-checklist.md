# Milestone Ritual Checklist

Relative path: `automation/checklists/milestone-ritual-checklist.md`
Project: Agent Bridge

Purpose: This checklist defines the boundary-review work that should happen at milestones in Agent Bridge. Use it when a seam boundary or phase boundary is reached so architecture, planning, and stabilized history updates happen together rather than drifting apart.

- Confirm a seam boundary or phase boundary was reached
- Gather relevant slice and checkpoint history
- Refresh architecture docs first
- Reconcile major risks and acceptance gaps
- Update roadmap, changelog, and developer journal for stabilized work
- Produce a milestone packet
- Record version journal entry
- Recommend milestone git tag or label
- Author the next phase or seam brief
