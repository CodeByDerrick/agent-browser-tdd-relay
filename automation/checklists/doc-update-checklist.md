# Doc Update Checklist

Relative path: `automation/checklists/doc-update-checklist.md`

Purpose: This checklist defines the baseline expectations for editing repo-authored docs in Agent Bridge. Use it to keep documentation compact, consistent, self-describing, and aligned with the workflow whenever a doc refresh happens.

- Prefer compact relay-specific language
- Preserve relay workflow rules and brief contract language
