# Checkpoint Ritual Checklist

Relative path: `automation/checklists/checkpoint-ritual-checklist.md`
Project: Agent Bridge

Purpose: This checklist defines the recurring maintenance work that should happen during a checkpoint in Agent Bridge. Use it whenever checkpoint reconciliation, documentation freshness, or boundary packaging is needed.

- Confirm checkpoint intent and current cadence context
- Gather recent slice reports when relevant and the current repo state
- Refresh affected docs from the watchlist
- Reconcile repo drift and process drift
- Review roadmap and backlog drift
- Produce a checkpoint packet
- Prepare commit recommendation
- Record version journal entry
- Update developer journal if checkpoint narrative matters
- Recommend checkpoint git tag or label
- Author the next minimal brief when the checkpoint changes the next action
