# Brief Title

Relative path: `automation/briefs/templates/task-brief-template.md`

Purpose: This template defines the standard shape for a minimal vertical-slice brief in `agent-browser-tdd-relay`. Use it when creating or refining a brief that is ready for slice ritual. It should contain only the information needed to run bounded TDD work with clear success criteria and evidence expectations.

## Slice

One minimal vertical slice.

## Red

Smallest failing test or observable failing condition that proves the slice matters.

## Green

Minimum behavior needed to satisfy the slice.

## Verification

The tests or checks that prove the slice works as intended after green.

## Refactor

Any limited cleanup that should happen only after verification and only if it improves clarity or reduces ongoing complexity.
