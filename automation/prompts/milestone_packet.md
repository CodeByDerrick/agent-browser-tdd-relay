# Milestone Packet Prompt

Relative path: `automation/prompts/milestone_packet.md`
Project: Agent Bridge

Purpose: This prompt frames the work of producing a milestone packet for seam or phase boundaries in Agent Bridge. Use it to summarize what stabilized, what remains risky, and which planning and history docs need updates when major work reaches a boundary worth recording.

Produce a milestone packet for the seam or phase boundary that was reached.

Requirements:

- refresh architecture docs before process summaries
- summarize what changed, what stabilized, and what remains risky
- recommend the next phase or seam brief
- include a milestone versioning recommendation
- include roadmap, changelog, and developer journal implications when the work is stabilized
