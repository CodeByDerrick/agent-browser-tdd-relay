# Reanchor Context Package Prompt

Relative path: `automation/prompts/reanchor_context_package.md`

Purpose: This prompt frames how to compress current relay work into a low-token resume package. Use it during reanchor ritual when context needs to survive a pause or handoff without dragging along unnecessary detail.

Compress the current work state into a low-token resume package.

Requirements:

- summarize current state
- list trusted docs/artifacts
- list the next action
- list drift risks
- include the roadmap/backlog/journal state that matters for restart
- keep the package short enough to reuse as restart context
